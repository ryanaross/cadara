## REMOVED Requirements

### Requirement: STEP assets SHALL retain original source bytes
**Reason**: Legacy STEP import removed; STEP asset retention will be respecified by the new STEP ImportProvider.
**Migration**: None required.

### Requirement: Baked mesh results SHALL be generated geometry assets
**Reason**: Legacy mesh import removed; mesh asset storage will be respecified by the new mesh ImportProvider.
**Migration**: None required.

### Requirement: Baked geometry assets SHALL record reconstruction quality metadata
**Reason**: Legacy mesh import removed; reconstruction metadata will be respecified by the new mesh ImportProvider.
**Migration**: None required.

### Requirement: Multi-file STEP imports SHALL retain every required source file
**Reason**: Legacy STEP import removed; multi-file asset retention will be respecified by the new STEP ImportProvider.
**Migration**: None required.

### Requirement: Multi-file STEP asset provenance SHALL preserve source names
**Reason**: Legacy STEP import removed; source name provenance will be respecified by the new STEP ImportProvider.
**Migration**: None required.
